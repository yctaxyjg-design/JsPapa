# 🗣️ 한글놀이 — 낱말 읽기 앱

6세 아이를 위한 **한글 낱말 소리 내어 읽기** 연습 앱입니다.
화면에 낱말이 크게 나오고, 아이가 마이크에 대고 읽으면 **발음을 인식해서 맞았는지 알려줍니다.**

## 기능

- **낱말 카드**: 동물 / 과일·채소 / 음식 / 물건·탈것 / 자연 / 가족·몸 — 6개 주제, 110여 개 낱말
- **🎤 말하기**: 아이가 읽으면 음성 인식으로 발음 확인 (자모 단위 유사도 비교라 "비슷해요!" 단계도 있음)
- **🔊 들어보기**: 정확한 발음을 기기 음성(TTS)으로 들려줌
- **🎁 힌트**: 이모지 그림 힌트
- **칭찬 보상**: 정답이면 색종이 + 효과음 + 칭찬 음성, 별⭐ 모으기 (localStorage 저장)
- **3번 시도**하면 정답 발음을 들려주고 다음 낱말로 넘어가서 좌절하지 않게 함
- 음성 인식이 안 되는 브라우저에서는 자동으로 **부모님 확인 모드**(⭕/🔁 버튼)로 전환

## 아이패드에서 쓰는 법

1. 이 폴더를 **HTTPS로 접속 가능한 곳에 배포**하세요. 가장 쉬운 방법은 GitHub Pages:
   - 저장소 → Settings → Pages → Branch를 `main`으로 선택
   - `https://<계정>.github.io/<저장소>/hangul/` 로 접속
2. 아이패드에서 **Safari로 접속** (음성 인식은 Safari에서만 동작, iOS 14.5 이상)
3. 처음 🎤 버튼을 누르면 **마이크 권한**을 묻습니다 → 허용
4. 공유 버튼 → **"홈 화면에 추가"** 하면 전체화면 앱처럼 사용 가능

> ⚠️ 음성 인식은 `file://`이나 HTTP에서는 동작하지 않습니다 (HTTPS 필수).
> 음성 인식은 Apple의 받아쓰기 서버를 사용하므로 인터넷 연결이 필요합니다.

## 로컬에서 개발할 때

```sh
python3 -m http.server 8000
# → http://localhost:8000/hangul/
```

`localhost`는 예외적으로 음성 인식이 허용됩니다 (데스크톱 Chrome/Safari 기준).

## 동작 원리

1. 낱말을 크게 표시
2. `webkitSpeechRecognition`(ko-KR)으로 아이 발음을 텍스트로 인식 (후보 최대 5개)
3. 목표 낱말과 인식 결과를 **자모 단위로 분해**해 Levenshtein 유사도 계산
   - 유사도 ≥ 0.8 → 정답 🎉
   - ≥ 0.5 → "아주 비슷해요, 한 번 더!"
   - 그 외 → "괜찮아요, 들어보고 다시!"
4. 짧은 낱말이 문장으로 인식되는 경우("곰이에요")도 포함 검사로 정답 처리

## 낱말 추가/수정

`words.js`의 `WORD_CATEGORIES` 배열에 `{ w: "낱말", e: "🐶" }` 형태로 추가하면 됩니다.
