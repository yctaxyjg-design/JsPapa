특정인 문자 자동 답장

지정한 사람의 iMessage/SMS에 내가 정한 말투와 분위기로 자동 답장하도록 단축어 오토메이션을 설정하세요.

← iPad 전원 스케줄러

먼저 알아두세요

iOS 정책상 모든 문자에 대한 무제한 자동응답은 불가능하고, 단축어 오토메이션의 "메시지" 트리거로 미리 지정한 사람에게만 자동 답장할 수 있습니다. 답장 생성에는 Claude API가 사용되므로 Anthropic API 키가 필요하며(사용량만큼 과금), 받은 메시지 내용이 API로 전송됩니다. 돈·약속 확정 같은 중요한 대화는 직접 답장하세요.

1. 자동 답장 대상 추가

2. 등록된 대상

  • 아직 등록된 대상이 없습니다.

3. 아이폰에서 설정하는 법

대상마다 아래 순서로 진행합니다: ① Anthropic API 키 준비 → ② 답장 생성 단축어 만들기 → ③ "메시지" 오토메이션 등록. 각 대상 카드의 프롬프트와 요청 본문을 복사해 사용하세요.

위에서 대상을 추가하면 맞춤 가이드가 표시됩니다.

이 페이지는 문자를 직접 보내지 않습니다. 설정은 브라우저(localStorage)에만 저장되며, API 키는 여기에 입력하지 마세요.
