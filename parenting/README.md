# 육아 공평분담 스프레드시트 (`parenting/`)

아빠·엄마 **두 사람이 함께 쓰는** 육아 분담 기록 웹앱.
FT 기사 「The perks of parenting by spreadsheet(스프레드시트로 육아하는 이점)」의 아이디어를
아이폰·아이패드 홈화면 앱(PWA) 형태로 옮겼습니다.

## 기능

- **탭 기록** — 매일 10개 육아 항목에서 실제 담당한 사람(`엄마` / `함께` / `아빠`)을 탭.
  같은 버튼을 다시 탭하면 해제. `함께`는 양쪽 0.5씩.
- **날짜 이동** — `‹ ›` 또는 날짜 선택으로 과거 기록 확인·수정.
- **누적 통계**
  - 누적 담당 횟수 (엄마 / 아빠)
  - 육아일 환산 격차 (하루 10개 항목 = 1육아일 기준)
  - 취침 담당 격차
  - 이번 달 분담률 원형 차트 (50:50 목표 대비)
- **이름 바꾸기** — `엄마`/`아빠` 라벨을 원하는 이름으로 변경.

### 육아 항목 (10종)

취침 전 동화 · 취침 · 아침식사 · 저녁식사 · 아이 등원 · 아이 하원 ·
평일 저녁 놀이 · 휴일 오전 놀이 · 휴일 오후 놀이 · 휴일 저녁 놀이

## iCloud Drive 공유 방식

순수 정적 웹앱이라 **실시간 자동 동기화(백엔드/CloudKit)는 제공하지 않습니다.**
대신 iCloud Drive 파일로 공유합니다.

1. 두 사람이 각자 홈화면 앱에서 기록한다.
2. **☁️ iCloud에 백업** → 공유 시트에서 "파일에 저장" → **공유 iCloud Drive 폴더**에 저장.
3. 상대가 **📥 불러오기 · 병합**으로 그 파일을 가져오면, **셀 단위로 병합**된다.
   같은 (날짜, 항목)은 더 최근에 기록한 값이 채택되므로, 둘이 각자 기록해도 합쳐집니다.

> 팁: 백업 파일 이름은 `육아분담_YYYY-MM-DD.json`. 같은 파일명을 덮어써 나가면
> 공유 폴더에 항상 최신본 하나만 유지됩니다.

## 데이터

- 이 기기의 브라우저 `localStorage`(`parenting-spreadsheet-v1`)에만 저장됩니다.
- 기기 간 이동/공유는 위 백업 파일로만 이루어집니다. 전체 삭제 전에는 백업을 권장합니다.

## 개발

빌드 도구·외부 의존성이 없는 정적 사이트입니다.

```sh
python3 -m http.server 8000
# → http://localhost:8000/parenting/
```
