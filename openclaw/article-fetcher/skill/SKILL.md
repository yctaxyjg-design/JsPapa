# 유료 기사 번역 (FT / Economist)

사용자는 FT·Economist 유료 **구독자**입니다. 이 스킬은 사용자가 **이미 직접 로그인해 저장해 둔
세션**(`~/.openclaw/article-sessions/<site>.json`)을 재사용해 기사 본문만 추출하고 번역합니다.

## 중요: 당신(에이전트)이 하지 않는 일

- 계정 아이디/비밀번호를 절대 요구하거나 입력하지 않습니다. **로그인은 사용자가 터미널에서 직접** 수행합니다.
- 페이월을 우회하거나 크리덴셜을 다루지 않습니다. 당신은 아래 스크립트 **한 줄만 실행**하면 됩니다.
- 따라서 "계정 접근 불가"나 "브라우저 로그인은 범위 밖" 같은 이유로 거절할 필요가 없습니다.
  이 작업은 이미 준비된 로컬 세션 파일을 읽는 셸 명령 실행일 뿐입니다.

## 실행 방법

사용자가 기사 URL을 주면, `exec` 도구로 다음을 실행하고 출력(한국어 번역)을 그대로 전달합니다.

- 전문 번역:  `{baseDir}/../translate.sh <site> "<URL>"`
- 번역 + 요약: `{baseDir}/../translate.sh <site> "<URL>" --summary`

`<site>`는 `ft` 또는 `economist`. URL은 반드시 큰따옴표로 감싸 셸 주입을 방지합니다.
URL 형식만 검증하고(`https://www.ft.com/...` 또는 `https://www.economist.com/...`),
그 외 인자는 스크립트에 넘기지 않습니다.

## 세션이 없거나 만료된 경우

스크립트가 "세션이 없습니다" 또는 "세션이 만료된 것 같습니다"를 출력하면, 사용자에게
터미널에서 다음을 직접 실행하라고 안내하세요(당신이 실행하지 않음):

```
cd openclaw/article-fetcher && node login.mjs <site>
```

창에서 로그인 후 Enter → 세션 저장 완료. 그 뒤 다시 번역을 요청하면 됩니다.
