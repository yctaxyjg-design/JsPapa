# JsPapa — iOS 단축어 자동화 도구 모음

## 특정인 문자 자동 답장 (`imessage/`)

지정한 사람의 iMessage/SMS에 내가 정한 말투·분위기로 자동 답장하도록 iOS 단축어 오토메이션을 설정하는 정적 웹앱입니다.

- 대상별로 말투(존댓말/반말), 답장 길이, 이모지 사용, 분위기 설명, 평소 답장 예시를 설정
- 설정에 맞는 **시스템 프롬프트**와 **Claude API 요청 본문(JSON)** 을 자동 생성 (복사 버튼 제공)
- 아이폰 단축어 앱에서 "메시지" 오토메이션 + `URL 콘텐츠 가져오기`로 등록하는 단계별 가이드 표시

### 한계

- iOS 정책상 모든 문자에 대한 무제한 자동응답은 불가능하며, **미리 지정한 발신자**에게만 작동합니다.
- 답장 생성에 Anthropic API 키가 필요합니다(사용량 과금). 받은 메시지가 API로 전송되므로 민감한 대화에는 사용하지 마세요.
- 설정은 브라우저 `localStorage`에만 저장됩니다. API 키는 웹앱에 입력하지 않고 단축어 안에만 넣습니다.

# iPad 전원 스케줄러

iPad의 전원을 시간대별로 켜고/끄는 일정을 관리하고, 그 일정을 iOS 단축어(Shortcuts) 오토메이션으로 설정하는 방법을 안내해주는 정적 웹앱입니다.

## 한계

iPadOS는 보안·전력 관리 정책상 **서드파티 앱이 기기의 전원을 직접 켜거나 끌 수 없습니다.** 따라서 이 앱은 다음과 같은 우회 자동화를 도와줍니다.

- 화면 잠금 / 깨우기
- 집중 모드(Focus) 전환
- 저전력 모드 전환
- 무음 / 볼륨 조절

## 사용법

1. `index.html`을 브라우저로 엽니다 (또는 GitHub Pages에 배포).
2. "일정 추가" 폼에서 이름, 동작, 시간, 요일을 입력합니다.
3. 등록된 일정 목록 아래에 단축어 앱 자동화 설정 가이드가 표시됩니다.
4. iPad의 단축어 앱에서 가이드대로 오토메이션을 등록하세요.

## 데이터

일정은 브라우저의 `localStorage`에만 저장됩니다. JSON으로 내보내기/가져오기가 가능합니다.

## 개발

빌드 도구가 없는 정적 사이트입니다. 파일을 직접 열거나 간단한 정적 서버로 띄우세요.

```sh
python3 -m http.server 8000
# → http://localhost:8000
```
