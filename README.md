# JsPapa

## 국기 맞히기 놀이 (`flag-game/`)

6세 아이를 위한 나라 국기 퀴즈 게임입니다. 국기를 보여주면 아이가 **말로** 나라 이름을 맞히고, 정답이면 그 나라의 특징·수도·역사 이야기를 음성으로 읽어줍니다.

- 음성 인식(Web Speech API, 한국어)으로 정답 판정 — 마이크가 안 되는 환경에서는 3지선다 버튼으로 대답
- 정답 시 색종이 축하 효과 + TTS로 나라 정보 낭독
- 한 번에 맞힌 문제마다 별점 ⭐, 10문제 한 판
- 아이패드 사파리에 최적화 (홈 화면에 추가하면 전체 화면 앱처럼 사용 가능)

`flag-game/index.html`을 아이패드 사파리로 열면 됩니다. 음성 인식은 보안 정책상 `https://` 또는 로컬 서버에서 열어야 동작합니다 (GitHub Pages 배포 추천).

## iPad 전원 스케줄러

iPad의 전원을 시간대별로 켜고/끄는 일정을 관리하고, 그 일정을 iOS 단축어(Shortcuts) 오토메이션으로 설정하는 방법을 안내해주는 정적 웹앱입니다.

## 한계

iPadOS는 보안·전력 관리 정책상 **서드파티 앱이 기기의 전원을 직접 켜거나 끌 수 없습니다.** 따라서 이 앱은 다음과 같은 우회 자동화를 도와줍니다.

- 화면 잠금 / 깨우기
- 집중 모드(Focus) 전환
- 저전력 모드 전환
- 무음 / 볼륨 조절

## 사용법

1. `index.html`을 브라우저로 엽니다 (또는 GitHub Pages에 배포).
2. "일정 추가" 폼에서 이름, 동작, 시간, 요일을 입력합니다.
3. 등록된 일정 목록 아래에 단축어 앱 자동화 설정 가이드가 표시됩니다.
4. iPad의 단축어 앱에서 가이드대로 오토메이션을 등록하세요.

## 데이터

일정은 브라우저의 `localStorage`에만 저장됩니다. JSON으로 내보내기/가져오기가 가능합니다.
