# 양도소득분 지방소득세 수납 대사 스크립트

경산시청 세무과의 양도소득분 지방소득세 업무에서 국세청 통보자료 엑셀과 지방세정보시스템 수납자료 엑셀을 납세자별로 대사하기 위한 실무용 스크립트입니다.

## 처리 로직 설계

1. **엑셀 읽기**
   - 국세청 통보자료와 지방세정보시스템 수납자료를 각각 읽습니다.
   - 주민등록번호, 사업자등록번호처럼 앞자리 `0`이 사라지면 안 되는 값이 있어 기본적으로 문자열로 읽습니다.
2. **컬럼 찾기**
   - 납세자 식별번호, 납세자명, 세액 컬럼을 컬럼명 키워드로 자동 탐색합니다.
   - 기관별 양식 컬럼명이 다르면 실행 옵션으로 정확한 컬럼명을 지정할 수 있습니다.
3. **자료 정제**
   - 납세자 식별번호의 하이픈, 공백, 불필요한 특수문자를 제거합니다.
   - `1,234,000`, `1,234,000원`, `(10,000)` 같은 문자형 숫자를 실제 숫자로 변환합니다.
4. **납세자별 합산**
   - 통보자료는 납세자별 통보세액으로 합산합니다.
   - 수납자료는 납세자별 수납세액으로 합산합니다.
5. **차이 계산 및 판정**
   - 차이금액은 `수납세액 - 통보세액`으로 계산합니다.
   - 차이금액이 ±2,000원 이하면 `정상`으로 판정합니다.
   - 통보세액은 있으나 수납세액이 없으면 `미납`으로 판정합니다.
   - 수납세액이 통보세액보다 2,000원 초과 부족하면 `과소`로 판정합니다.
   - 수납세액이 통보세액보다 2,000원 초과 많으면 `환급`으로 판정합니다.
6. **엑셀 출력**
   - 전체 결과, 판정별 시트, 요약 시트를 하나의 결과 엑셀 파일로 저장합니다.

## 사용 방법

필요 패키지를 설치합니다.

```bash
python3 -m pip install pandas openpyxl
```

기본 실행 예시는 다음과 같습니다.

```bash
python3 scripts/tax/reconcile_local_income_tax.py \
  --notice data/국세청_통보자료.xlsx \
  --payment data/지방세정보시스템_수납자료.xlsx \
  --output output/양도소득분_지방소득세_대사결과.xlsx
```

컬럼 자동 탐색이 실패하거나 같은 의미의 컬럼이 여러 개 있으면 컬럼명을 직접 지정합니다.

```bash
python3 scripts/tax/reconcile_local_income_tax.py \
  --notice data/국세청_통보자료.xlsx \
  --payment data/지방세정보시스템_수납자료.xlsx \
  --notice-id-col 주민등록번호 \
  --notice-name-col 납세자명 \
  --notice-amount-col 지방소득세액 \
  --payment-id-col 주민등록번호 \
  --payment-name-col 납세자명 \
  --payment-amount-col 수납액 \
  --output output/양도소득분_지방소득세_대사결과.xlsx
```

다음 달에도 같은 명령에서 파일명만 바꾸어 재사용하면 됩니다.

## 오류 가능성 체크

- **컬럼명 불일치**: 엑셀 양식의 컬럼명이 예상 키워드와 다르면 `--notice-*-col`, `--payment-*-col` 옵션으로 직접 지정합니다.
- **식별번호 누락 또는 변형**: 주민등록번호 앞자리 `0` 누락, 하이픈 포함 여부, 엑셀의 지수 표기 변환 여부를 확인합니다.
- **동명이인 문제**: 기본 키는 `납세자식별번호 + 납세자명`입니다. 식별번호가 비어 있으면 동명이인이 합쳐질 수 있으므로 원자료를 보정해야 합니다.
- **세액 컬럼 선택 오류**: 지방소득세 본세, 가산세, 합계액 등 유사 컬럼이 여러 개면 실제 대사 기준 컬럼을 명시합니다.
- **환급/충당 자료 반영 시점 차이**: 수납자료 기준일과 국세청 통보자료 기준일이 다르면 정상 건도 차이가 날 수 있습니다.
- **음수 금액 표기**: 괄호 음수 `(10,000)`은 `-10000`으로 처리합니다. 다른 특수 표기가 있으면 사전 점검이 필요합니다.
- **중복 행**: 납세자별로 합산하므로 분납 자료는 처리되지만, 원자료 중복 추출분은 결과도 중복 반영됩니다.

## 실무용 설명문

본 대사표는 국세청 통보자료의 양도소득분 지방소득세 통보세액과 지방세정보시스템 수납자료의 실제 수납세액을 납세자별로 합산한 뒤, 수납세액에서 통보세액을 뺀 차이금액을 기준으로 판정한 자료입니다. 차이금액이 ±2,000원 이내인 경우에는 소액 차이로 보아 정상으로 처리하며, 통보세액이 있으나 수납세액이 없는 경우에는 미납, 수납세액이 부족한 경우에는 과소, 수납세액이 더 많은 경우에는 환급 검토 대상으로 분류합니다. 결과 파일의 `전체결과` 시트에서 전체 내역을 확인하고, `미납`, `과소`, `환급` 시트는 후속 안내, 부과 또는 환급 검토 대상 목록으로 활용할 수 있습니다. 최종 처분 전에는 원자료 기준일, 납세자 식별번호, 세액 컬럼, 중복 추출 여부를 반드시 확인해야 합니다.
