# 🎨 수학놀이터 — 애플펜슬로 배우는 수 개념 (6세)

단순 연산 반복이 아니라 **수 개념의 원리**를 애플펜슬 필기로 익히는 아이패드용 웹앱입니다.
아이패드 사파리에서 열고 **공유 → 홈 화면에 추가**하면 전체 화면 앱처럼 사용할 수 있습니다.

## 놀이 구성과 교육 이론 배경

| 놀이 | 하는 일 | 배우는 개념 | 이론 배경 |
|------|---------|-------------|-----------|
| 🍓 세어 보고 쓰기 | 그림을 콕콕 눌러 세고 펜으로 숫자 쓰기 | 일대일 대응 세기, 숫자 형태 | 몬테소리 모래 숫자판(따라 쓰기 안내 포함) |
| ✨ 번쩍! 몇 개? | 잠깐 나타난 점 패턴의 개수를 한눈에 맞히기 | 직산(subitizing), 5 기준 수 감각 | Number Talks의 점 패턴(dot cards), 주사위 배열 |
| 🍎 사과 가르기 | 펜으로 선을 그어 사과를 두 무리로 가르기 | 수 가르기·모으기, `n = a + b` | 초등 1학년 "가르기와 모으기", 싱가포르 수학 Number Bond |
| 🐣 10 만들기 | 십틀의 빈 칸 수를 세어 쓰기 | 10의 보수, 받아올림의 기초 | Ten Frame, Make-10 전략 |
| 🍪 똑같이 나눠요 | 쿠키에서 접시까지 선을 그어 공평하게 분배 | 등분(나눗셈의 원개념), 공평성 | Fair Sharing(등분제) 연구, CPA(구체물→그림→기호) 접근 |

공통 설계 원칙은 DragonBox류 개념 학습 게임과 같습니다 — **기호보다 개념 먼저, 정답 채점보다 조작 경험 먼저.**
모든 문제는 한국어 음성(TTS)으로 읽어 주므로 글을 몰라도 혼자 놀 수 있습니다.

## 애플펜슬 필기 인식

- 캔버스에 쓴 획을 **$P Point-Cloud Recognizer** (Vatavu, Anthony & Wobbrock, ICMI 2012)로 인식합니다.
  점 구름 매칭이라 획 순서·방향에 둔감해 아이의 자유로운 필체에 강합니다. (`recognizer.js`, 외부 의존성 없음)
- 숫자 0~9마다 여러 벌의 템플릿을 내장했고, 인식 거리가 큰 낙서는 "잘 모르겠어요"로 처리해 다시 쓰게 합니다.
- 펜(Apple Pencil) 입력이 감지되면 손바닥 터치를 무시하는 **팜 리젝션**이 켜집니다. 펜이 없으면 손가락으로도 됩니다.

## 실행

빌드 도구가 없는 정적 사이트입니다.

```sh
cd math-play
python3 -m http.server 8000
# → http://localhost:8000
```

GitHub Pages 등에 올리면 아이패드에서 바로 접속할 수 있습니다.
음성은 iOS 설정에 한국어 Siri 음성이 있으면 그 목소리로 나옵니다.

## 인식기 자가 테스트

`recognizer.js`는 외부 의존성이 없어 Node에서 바로 검증할 수 있습니다.

```sh
node math-play/recognizer.test.js
# 내장 숫자 템플릿이 자기 숫자로(거리 < 1.34) 인식되는지,
# 빈 입력·낙서가 걸러지는지 확인 → "43 passed, 0 failed"

node math-play/palm-rejection.test.js
# 팜 리젝션: 긴 필기(>700ms) 직후의 손바닥 터치도 걸러지는지 등 → "6 passed, 0 failed"
```

## 파일 구성

- `index.html` — 진입점 (PWA 메타 포함)
- `app.js` — 게임 5종, 잉크 패드(포인터 이벤트), TTS/효과음/폭죽, 별 저장(localStorage)
- `recognizer.js` — $P 손글씨 숫자 인식기 (외부 의존성 없음, Node에서도 로드 가능)
- `recognizer.test.js` — 인식기 자가 테스트 (Node 실행)
- `palm-rejection.js` — 팜 리젝션 상태 머신 (순수 로직, Node에서도 로드 가능)
- `palm-rejection.test.js` — 팜 리젝션 회귀 테스트 (Node 실행)
- `styles.css`, `manifest.webmanifest`, `icon.svg`, `icon-192.png`, `icon-512.png`
